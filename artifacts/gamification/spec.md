# 게임화 (잔디·관심사 배지·레벨)

## 개요
학습을 계속하고 싶게 만드는 게임화 요소를 홈 화면에 추가한다. 날짜별 학습량을 잔디(캘린더 히트맵)로 보여주고, 많이 학습한 주제에는 배지를 자동으로 주고, 누적 점수에 따라 레벨이 오른다. 문제 생성·채점 엔진은 건드리지 않는다.

## 범위

### 포함
- 최근 1년간 날짜별 학습량을 색 강도로 보여주는 잔디 캘린더
- 특정 주제(태그)를 일정 문항 수 이상 풀면 자동으로 그 주제 배지를 획득
- 누적 점수 기반 레벨과 다음 레벨까지 진행률 표시
- 세트 완료 직후 결과 화면에 새로 얻은 배지·레벨업 알림
- 위 세 가지 모두 홈 화면에 통합 표시

### 제외
- 랭킹·리더보드 등 다른 사용자와의 비교 — "나 혼자 MVP" 원칙 유지, 이번 범위 아님
- 배지 단계(브론즈/실버/골드 등) — 태그당 배지는 하나만 존재하도록 단순화 (합의됨)
- 이 기능 도입 이전에 완료된 세션의 소급 반영 — 태그별 학습 횟수를 지금까지 저장해오지 않아 정확한 소급 계산이 불가능하고, 사용자도 깔끔하게 새로 시작하는 쪽을 선택함
- 레벨 상한 — 무제한으로 계속 오름 (제안 기본값)
- 문제 생성·채점(Gemini 호출) 로직 변경 — 사용자가 명시적으로 이번 범위에서 제외 요청

## 시나리오

### S1. 홈 화면의 잔디 캘린더
- **Given**: 이 기능이 도입된 이후 학습 기록이 있는 상태에서 홈 화면에 진입
- **When**: 홈 화면을 본다
- **Then**
  - [x] S1-1 최근 1년(오늘 포함) 날짜가 달력 히트맵 형태로 표시된다
  - [x] S1-2 그 날 푼 문항 수가 많을수록 더 진한 색으로 표시된다(여러 단계의 색 강도)
  - [x] S1-3 학습하지 않은 날은 색이 채워지지 않은 빈 칸으로 표시된다
  - [x] S1-4 이 기능 도입 이전에 완료된 세션은 잔디에 반영되지 않는다

### S2. 세트 완료가 잔디에 반영됨
- **Given**: 오늘 아직 잔디가 채워지지 않은 상태(또는 이미 일부 채워진 상태)
- **When**: 오늘 세트 하나를 끝까지 풀어 완료한다
- **Then**
  - [x] S2 홈 화면에 재진입하면 오늘 날짜 칸이 그 세트의 문항 수만큼(같은 날 여러 세트 완료 시 합산) 갱신된 색 강도로 표시된다

### S3. 관심사 배지 자동 획득
- **Given**: 이 기능 도입 이후 특정 주제 태그의 문항을 9개까지 푼 상태
- **When**: 같은 태그의 문항을 1개 더 풀어(총 10개) 세트를 완료한다
- **Then**
  - [ ] S3-1 세트 완료 직후 결과 화면에 "새 배지 획득: {주제}"(제안 기본값 문구) 알림이 나타난다
  - [ ] S3-2 이후 홈 화면에 그 주제 배지가 계속 표시된다

### S4. 배지 미달성 주제
- **Given**: 이 기능 도입 이후 특정 주제 태그의 문항을 9개 이하로 푼 상태
- **When**: 홈 화면을 본다
- **Then**
  - [ ] S4 그 주제의 배지는 아직 표시되지 않는다

### S5. 홈 화면의 배지 목록
- **Given**: 배지를 하나도 획득하지 못한 상태이거나, 하나 이상 획득한 상태
- **When**: 홈 화면을 본다
- **Then**
  - [ ] S5-1 획득한 배지가 있으면 목록으로 표시된다
  - [ ] S5-2 하나도 없으면 "아직 획득한 배지가 없습니다"(제안 기본값)가 표시된다

### S6. 홈 화면의 레벨 표시
- **Given**: 이 기능 도입 이후 누적 점수가 쌓인 상태
- **When**: 홈 화면을 본다
- **Then**
  - [ ] S6-1 현재 레벨(누적 점수 500점당 1레벨, 1레벨부터 시작 — 제안 기본값)이 표시된다
  - [ ] S6-2 다음 레벨까지 남은 진행률이 표시된다

### S7. 레벨업 알림
- **Given**: 세트를 완료하면 누적 점수가 다음 레벨 경계(500점 배수)를 넘기게 되는 상태
- **When**: 그 세트를 완료한다
- **Then**
  - [ ] S7 세트 완료 직후 결과 화면에 "레벨업!"(제안 기본값) 알림이 나타난다

## 불변 규칙

- [ ] INV-1 잔디·배지·레벨은 이 기능이 도입된 시점 이후 완료된 세션만 반영한다. 그 이전 세션 기록은 반영되지 않는다
- [ ] INV-2 홈 화면에 표시되는 레벨·배지·잔디는 항상 저장된 세션 완료 기록으로부터 계산한 값과 일치한다(새로고침·재방문해도 값이 바뀌지 않는다)

## End-to-end 검증

개발 서버(`bun run dev`)를 띄우고 진행한다. 이 기능 도입 이후 시점 기준으로 테스트한다.

1. 홈 화면 진입 → 아직 활동이 없으면 빈 잔디, 레벨 1, "아직 획득한 배지가 없습니다" 확인 (S1, S5-2, S6-1)
2. 특정 주제 태그의 문항이 포함된 세트를 반복 완료해 그 태그를 10문항 이상 푼다 → 10번째를 완료하는 세트의 결과 화면에서 배지 획득 알림 확인 (S3-1) → 홈 화면에서 배지 표시 확인 (S3-2, S5-1)
3. 세트를 반복 완료해 누적 점수가 500점을 넘기게 한다 → 그 순간 결과 화면에서 레벨업 알림 확인 (S7) → 홈 화면에서 레벨 2·진행률 확인 (S6)
4. 오늘 여러 세트를 완료한 뒤 홈 화면에서 오늘 날짜 잔디 칸의 색 강도가 문항 수 합계를 반영하는지 확인 (S1-2, S2)
5. 페이지 새로고침 → 잔디·배지·레벨이 그대로 유지되는지 확인 (INV-2)

## 의존성
